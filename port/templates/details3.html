Project description

Online examination system developed using PHP, worked in a group of four.

User types

Admin

Who are in charge of performing modifications to all users

Teacher

Who are assigned to teach courses and set tests

Student

Who enroll courses and take tests

Features and functions

Admin

View all registered accounts, register new teachers and admins, view courses taght by teachers and available courses for a specific teacher to teach, enrol teachers to teach a course, view records for each student (info, courses enrolled, test taken and marks), view all registered departments, programs, and courses, add new departments, programs and courses

Teacher

View the courses which they are teaching, set tests for courses, view all the tests set for a specific course, view students of a course, view test and mark of each student

Student

Register an account, view enrolled courses, view available tests from courses, take tests, enroll courses, view test marks
